## 파일명에 대하여.

### etc 폴더

- 기본적인 파일명은 **(백준의 번호)**.py 입니다.
- 번호 뒤에 _notcomp 가 붙은 것은 완성되지 않은 코드입니다.
- 이외에도 몇몇 단어가 붙어있으나 뭔가 부정적이다 싶은 것은 안풀린 것이고, -ing 같은 경우는 try중인 문제(아마도?), done 같은 것이 붙어있다면 예전에 안풀려서 버려뒀다가 나중에 푼 문제 입니다.
